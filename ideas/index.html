20 Vibe‑Coding Game Ideeën

- De score van het spel kan gesubmit worden naar een API.
- De documentatie van de API staat op:
  https://score-controller.vercel.app/docs.html

Deze games kun je binnen één lesuur bouwen met hulp van een AI‑tool zoals Claude of Copilot. Ze zijn klein, duidelijk en ideaal om programmeerlogica en prototyping te oefenen.

Game‑ideeën

1. Number Guessing Game
  • Speler raadt een geheim nummer.
  • Opties: hoger/lager hints, beperkt aantal pogingen.
  • Multiplayer-variant mogelijk.
2. Simon Says
  • Kleurvolgorde onthouden en reproduceren.
  • Elke ronde komt er een kleur bij.
  • Perfect voor DOM‑interactie.
3. Endless Dodger
  • Vallende objecten ontwijken.
  • Score stijgt naarmate je langer overleeft.
  • Makkelijk met basis-animaties.
4. Reflex Math / Typing Trainer
  • Som of woord tonen, speler moet snel reageren.
  • Tijdslimiet maakt het spannend.
  • Highscore‑mechaniek mogelijk.
5. Dobbelsteen-Battle
  • Speler vs computer: wie rolt het hoogst?
  • Opties: verschillende soorten dobbelstenen.
  • Animaties voor extra flair.
6. Memory Game
  • Memory met emoji's of plaatjes.
  • 6–10 kaarten voor snelle gameplay.
  • Match = kaart blijft open.
7. Quick Draw Duel
  • Wacht op signaal → klik zo snel mogelijk.
  • 1 of 2 spelers.
  • Random delays voegen spanning toe.
8. Tamagotchi-Lite
  • Eenvoudige statusbalken zoals honger/energie.
  • Waarden dalen automatisch.
  • Speler moet knoppen gebruiken om het karakter te houden leven.
9. Target Clicker Game
  • Doelen verschijnen willekeurig.
  • Speler moet razendsnel klikken.
  • Mis = einde spel.
10. Mini Music Sequencer
  • Grid waar je geluidjes kunt activeren.
  • Loop speelt automatisch af.
  • Intro tot simpele audio‑API’s.
11. Reaction Maze
  • Doolhof met obstacles.
  • Je moet het doel binnen X seconden bereiken.
  • Random gegenereerde layouts.
12. Emoji Slot Machine
  • Drie draaiende emoji‑rollen.
  • Matching combinaties geven punten.
  • Zeer visueel en simpel.
13. Flappy‑Bird‑Lite
  • Druk op één knop om te vliegen.
  • Obstakels bewegen richting speler.
  • Supergoed te bouwen in canvas.
14. Battle Quiz Game
  • Goed antwoord: tegenstander verliest HP.
  • Fout antwoord: jij verliest HP.
  • Lichte RPG‑mechanieken.
15. Rock–Paper–Scissors Deluxe
  • Uitgebreide versie met extra opties.
  • Lizard & Spock toevoegen mogelijk.
  • Leuk voor animaties.
16. Platform Jump Game
  • Spring zo hoog mogelijk via platforms.
  • Platforms spawnen random.
  • Score = hoogte.
17. Mini Escape Room
  • 4–5 mini‑puzzels op één pagina.
  • Codes, sliders, verborgen knoppen.
  • Toegankelijk en creatief.
18. Traffic Light Challenge
  • Groen = klik, rood = niet klikken.
  • Vals signaal maakt het pittiger.
  • Perfect voor reaction‑training.
19. Word Builder Game
  • Random letters → maak zoveel mogelijk woorden.
  • 30‑secondentimer.
  • Gebruik kleine lijst voor woordcontrole.
20. Collect the Coins
  • Beweeg door een speelveld en verzamel munten.
  • Tijdslimiet maakt het spannend.
  • Heel simpel via DOM‑positie.
