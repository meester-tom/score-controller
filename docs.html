📘 Score Submit API — Documentatie

Deze API laat je een score insturen vanuit je app. Je stuurt een naam en een reactietijd, en de server slaat dit op.

📍 Endpoint

POST https://score-controller.vercel.app/api/submit-score

📦 Request body (JSON)

{
  "name": "string",
  "score": 0.123
}

name — de naam van de speler
score — de reactietijd in seconden (nummer)

🧪 Voorbeeld request

fetch("https://score-controller.vercel.app/api/submit-score", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    name: "Sanne",
    score: 0.241
  })
});

📥 Response bij succes

{
  "success": true,
  "entry": {
    "name": "Sanne",
    "score": 0.241,
    "timestamp": 1736862342342
  }
}

⚠️ Response bij fout

{ "error": "Invalid payload" }

📌 Regels

  • Gebruik altijd POST
  • Stuur JSON met name en score
  • Score moet een nummer zijn
