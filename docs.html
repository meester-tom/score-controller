📘 Score Submit API — Documentatie

Deze API laat je scores (punten) opslaan en uitlezen. Je stuurt een naam en score, en de server bewaart dit tijdelijk in-memory.

📍 Endpoint

https://score-controller.vercel.app/api/submit-score

🧭 Ondersteunde methods

  • POST — voeg een score toe
  • GET — haal alle scores op
  • DELETE — verwijder alle scores
  • OPTIONS — CORS preflight

📦 POST request body (JSON)

{
  "name": "string",
  "score": 123
}

name — de naam van de speler
score — aantal punten (nummer)

🧪 Voorbeeld POST request

fetch("https://score-controller.vercel.app/api/submit-score", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    name: "Sanne",
    score: 241
  })
});

📥 Responses bij succes

POST

{
  "success": true,
  "entry": {
    "name": "Sanne",
    "score": 241,
    "timestamp": 1736862342342
  }
}

GET

[
  {
    "name": "Sanne",
    "score": 241,
    "timestamp": 1736862342342
  }
]

DELETE en OPTIONS

{ "success": true }

⚠️ Responses bij fouten

  • { "error": "Invalid payload" } (400)
  • { "error": "Origin not allowed" } (403)
  • { "error": "Method not allowed" } (405)
  • { "error": "Server error" } (500)

🌐 CORS

Cross-origin requests zijn toegestaan voor:

  • https://*.vercel.app
  • http://localhost:* en https://localhost:*
  • http://127.0.0.1:* en https://127.0.0.1:*
  • Losse browserbestanden zonder server (Origin: null, bijvoorbeeld file://)

Andere origins krijgen een 403 response.

📌 Regels

  • Stuur bij POST JSON met name en score
  • Score moet een nummer zijn
  • Data staat in-memory en kan resetten als de serverless functie opnieuw opstart
